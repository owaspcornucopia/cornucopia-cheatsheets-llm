# C5 — Influence Trust in the Application

## Non-Applicable

This threat does not apply to this application.

## Reasoning

The C5 card describes influencing the trust other parties have in the application or abusing that trust elsewhere.

This application is an internal API tool consumed programmatically by other systems. It is not:

- A public-facing website where user trust matters for engagement
- A branded consumer service where reputation drives business
- A platform that issues certificates or attestations trusted by third parties
- An identity provider whose trust assertions are relied upon by others

Trust manipulation (phishing using the application's brand, impersonating the application to trick users, undermining confidence in the service) is not a primary attack vector for a backend fraud investigation API consumed by a small set of known API clients.
