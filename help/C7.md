# C7 — Inadequate Audit Trail

## Non-Applicable

This threat does not apply to this application.

## Reasoning

The C7 card describes actions that cannot be investigated because there is no adequate audit trail, no full record of security events, or because logs can be altered or deleted.

While the logging concern is valid for this application, it is already comprehensively covered by the Ace (A) cards in each category:

- VEA — cannot detect attacks against input validation
- ATA — cannot detect authentication attacks
- SMA — cannot detect session management attacks
- AZA — cannot detect authorization bypass

Adding C7 as a separate applicable threat would be redundant. The absence of logging and auditability is the same underlying issue described from four different angles by those cards.
